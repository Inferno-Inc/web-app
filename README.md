## Server Configuration

### NGINX

```nginx
server {
    listen      127.0.0.1:80;
    server_name secu.int;
    
    rewrite ^/(.*)/$ /$1 permanent;

    root `{/path/to/secu/web-app}`/public;
    index index.html;

    location ~* \.(jpg|jpeg|gif|png|ico|css|pdf|ppt|txt|bmp|rtf|js)$ {
        expires 7d;
    }

    location / {
        try_files $uri $uri/ /index.html?$query_string;
    }
}
```
